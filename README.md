# excel-row-duplication-remover

## Requirements
  * Python 3.5+

## Installation
  1. Activate virtual environment (If any)
  2. Install requirements
```
pip install -r requirements.txt
```

## Usage
```
cd excel-row-duplication-remover
python remover.py [FILE NAME] --column [COLUMN TO CHECK]
```

## Help
```
python remover.py -h
```
